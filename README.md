# AudioBoard

Im Ordner Hardware befindet sich der Schaltplan als KiCAD Projekt und die Gerberdaten der Platine.

Die Software gibt es als "Basic" Version, welche die Grundlegenden Funktionen zur verfügung stellt. Auf der Basis von diesen Daten können eigene Projekte entwickelt werden.
Der Quelltext zum Matrixprojekt ist auf genau dieses Projekt zugeschnitten. Es wird dazu eine 7x6 Matrix mit WS2812 LEDs erwartete, welche alle seriell verbunden sind. Dieser Quelltext soll eher als Inspiration dienen.
